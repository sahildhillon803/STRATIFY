# Contributing to STRATIFY

First off, thank you for considering contributing to STRATIFY! It's people like you that make STRATIFY a great tool for startups worldwide.

## Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct. Please read [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) to understand the expectations.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue in the GitHub repository. Include:
1. Steps to reproduce the bug.
2. Expected behavior.
3. Actual behavior.
4. Your operating system and browser version.

### Suggesting Enhancements
Have an idea to make STRATIFY better? We'd love to hear it. Submit an issue detailing your suggestion, why it's valuable, and how it should work.

### Submitting Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **If you've added code** that should be tested, add tests.
3. **Ensure the test suite passes**.
4. **Make sure your code lints** (e.g., using `flake8` for Python, `eslint` for JS/TS).
5. **Issue that pull request!**

## Development Setup
Please refer to the `Installation Guide` in our [README.md](README.md) for instructions on setting up the project locally.

We look forward to your contributions!
